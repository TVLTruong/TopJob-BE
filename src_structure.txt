TopJob BE - Mô tả chức năng từng folder/file trong src

src/
- app.controller.spec.ts: Test cho AppController (mẫu NestJS).
- app.controller.ts: Controller gốc, có thể cung cấp route kiểm tra (ping/health).
- app.module.ts: Root module, import ConfigModule, TypeOrmModule và các feature modules.
- app.service.ts: Service gốc, cung cấp logic dùng bởi AppController (nếu có).
- main.ts: Bootstrap ứng dụng Nest, cấu hình global pipes/filters/interceptors nếu có.

- common/: Tiện ích/cấu phần dùng chung toàn app
  - decorators/: Custom decorators cho controller/route
    - current-user.decorator.ts: Trích xuất user từ request.
    - public.decorator.ts: Đánh dấu route không yêu cầu auth.
    - roles.decorator.ts: Gắn metadata vai trò cho route.
  - enums/: Hằng số dạng enum
    - company-size.enum.ts: Enum kích thước công ty.
    - user-role.enum.ts: Enum vai trò user (candidate/employer/admin).
    - user-status.enum.ts: Enum trạng thái user (pending/active/banned).
  - filters/
    - http-exception.filter.ts: Bắt/chuẩn hoá lỗi HTTP trả về client.
  - guards/: Bảo vệ route
    - jwt-auth.guard.ts: Guard xác thực JWT.
    - roles.guard.ts: Guard kiểm tra quyền/role dựa metadata.
  - interceptors/
    - transform.interceptor.ts: Chuẩn hoá/bao gói response trả về.
  - interfaces/
    - request-user.interface.ts: Kiểu dữ liệu user đính kèm trong request.
  - pipes/
    - validation.pipe.ts: Pipe validate DTO sử dụng class-validator.

- config/: Cấu hình hệ thống
  - cloudinary.config.ts: Cấu hình Cloudinary upload.
  - database.config.ts: Thiết lập TypeORM (Postgres, synchronize=false, autoload entities).
  - jwt.config.ts: Thiết lập JWT (secret, expires).

- database/: Cấu hình nguồn dữ liệu
  - data-source.ts: Khởi tạo DataSource cho TypeORM.

- modules/: Các module nghiệp vụ chính
  - applications/: Quản lý ứng tuyển
    - applications.controller.ts: Endpoints nộp/duyệt/tra cứu application.
    - applications.module.ts: Module wiring (controller, service, entity).
    - applications.service.ts: Nghiệp vụ ứng tuyển (tạo, cập nhật trạng thái, truy vấn).
    - dto/: (hiện chưa có file)
    - entities/
      - application.entity.ts: Entity bảng applications (candidate, job, trạng thái...).

  - auth/: Xác thực/đăng nhập/đăng ký
    - auth.controller.ts: Endpoints đăng ký/đăng nhập/xác thực, gửi token.
    - auth.module.ts: Module wiring cho auth.
    - auth.service.ts: Nghiệp vụ đăng nhập, mã hoá mật khẩu, phát hành token.
    - dto/
      - login.dto.ts: Payload đăng nhập.
      - register.dto.ts: Payload đăng ký.
    - entities/
      - email-verification-token.entity.ts: Entity token xác minh email.
      - password-reset-token.entity.ts: Entity token đặt lại mật khẩu.

  - candidates/: Hồ sơ ứng viên
    - candidates.controller.ts: Endpoints CRUD ứng viên, cập nhật hồ sơ.
    - candidates.module.ts: Module wiring cho candidates.
    - candidates.service.ts: Nghiệp vụ hồ sơ ứng viên.
    - dto/
      - create-candidate.dto.ts: DTO tạo hồ sơ.
      - update-candidate.dto.ts: DTO cập nhật hồ sơ.
    - entities/
      - candidate.entity.ts: Entity bảng candidates (thông tin cá nhân, liên kết user).

  - categories/: Danh mục chung
    - categories.controller.ts: Endpoints danh mục (placeholder).
    - categories.module.ts: Module wiring cho categories.
    - categories.service.ts: Logic danh mục (placeholder).

  - company-categories/: Danh mục ngành công ty
    - entities/
      - company-category.entity.ts: Entity bảng companies_categories.

  - employers/: Nhà tuyển dụng/công ty
    - employers.controller.ts: Endpoints CRUD nhà tuyển dụng.
    - employers.module.ts: Module wiring cho employers.
    - employers.service.ts: Nghiệp vụ doanh nghiệp.
    - dto/
      - create-employer.dto.ts: DTO tạo employer.
      - update-employer.dto.ts: DTO cập nhật employer.
    - entities/
      - employer.entity.ts: Entity bảng employers (liên kết user, thông tin công ty, trạng thái duyệt...).
      - employer-location.entity.ts: Entity bảng employer_locations (văn phòng/địa điểm công ty).

  - job-categories/: Danh mục ngành nghề tuyển dụng
    - entities/
      - job-category.entity.ts: Entity bảng jobs_categories.

  - jobs/: Tin tuyển dụng
    - jobs.controller.ts: Endpoints CRUD tin tuyển dụng, publish/close.
    - jobs.module.ts: Module wiring cho jobs.
    - jobs.service.ts: Nghiệp vụ tin tuyển dụng (tạo, lọc, thống kê...).
    - dto/: (hiện chưa có file)
    - entities/
      - job.entity.ts: Entity bảng jobs (lương, kỹ năng, vị trí, quan hệ...).

  - saved-jobs/: Tin đã lưu của ứng viên
    - entities/
      - saved-job.entity.ts: Entity bảng saved_jobs (ứng viên lưu tin).

  - upload/: Upload tệp (ví dụ avatar/CV)
    - upload.controller.ts: Endpoints upload file tới Cloudinary.
    - upload.module.ts: Module wiring cho upload.
    - upload.service.ts: Tương tác Cloudinary, trả về URL.

  - users/: Tài khoản người dùng
    - users.controller.ts: Endpoints CRUD user, quản lý profile cơ bản.
    - users.module.ts: Module wiring cho users.
    - users.service.ts: Nghiệp vụ user (tạo user, tìm theo email, v.v.).
    - dto/
      - create-user.dto.ts: DTO tạo user.
      - update-user.dto.ts: DTO cập nhật user.
    - entities/
      - user.entity.ts: Entity bảng users (email, password_hash, role, status...).

Ghi chú:
- TypeORM đang cấu hình synchronize=false; cần migrations hoặc bật sync ở dev để tạo schema.
- Một số thư mục DTO có thể bổ sung thêm trong tương lai khi mở rộng nghiệp vụ.
